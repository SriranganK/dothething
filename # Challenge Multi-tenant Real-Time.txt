# Challenge: Multi-tenant Real-Time Project Operations Platform

Think of it as a mix of Jira + Slack + GitHub + analytics.

You build it once:

1. React + Node.js + MongoDB
2. React + Go + MongoDB
3. React + ASP.NET + MongoDB (or SQL if you want an extra challenge)

This lets you compare backend design, concurrency, authentication, and scaling approaches.
## Core features

### 1. Multi-tenant organization system

Users can:

* Create organizations
* Invite members
* Join teams
* Assign roles

Roles:

* Owner
* Admin
* Project manager
* Developer
* Viewer

Backend tasks:

* RBAC authorization
* Organization isolation
* Permission middleware
* Invitation tokens
* Audit logs

Frontend tasks:

* Dynamic navigation based on role
* Protected routes
* Permission-based UI rendering
